# Optimize Workflow runs

Create a plan to overhaul the entire set of workflow runs to optimize and then execute.

## Dependencies

- Gemini MCP server tool accesible and working.
- Can update code
- Can create branches, issues, PR, and reply and resolve PR review comments.
- Create your new branch and checkout to switch to it.

## Problems

- Organically created. Needs a explicit strategy
- Many steps taker > 20-30 minutes.
- GitHub cacehing strategy unimplemented or weakly so.

## Plan

- Collaborate with Gemini. (Use you gemini-mcp brainstorming or appropraite tool) to identify problems and get overall feedback.
- Based on review feedback, create a detailed step-by-step plan with todo-list.
- Add explanations and rationality.

## Review Plan

- Review the plan for feedback with Gemini-MCP.
- Provide feedback.

## Apply Plan Review feedback
 
- Optimze and update plan based on the plan review.
- Create GitHub issue documenting plan.

## Execute Plan

- Once plan is crated, reviewed, and oprimized. Apply thr plan.
- Create a new branch and PR based on issue created in last step.
- Commit changes to the PR

## Review Changes

- Once changes have been applied, review the changes in the PR.
- Have Gemini and yourself conduct separate PR reviews.

## Apply Review Changes

- Once PR reviews are finished, collect list of PR review comments.
- Create a detailed plan to resolved the comments.
- Exceute plan, update each PR review comment summary of fix and then resolved that PR review thread.
